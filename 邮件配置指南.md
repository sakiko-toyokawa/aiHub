# QQ 邮箱 SMTP 配置指南

## 📧 什么是 SMTP？

SMTP（简单邮件传输协议）用于发送邮件。配置后，AI Knowledge Hub 可以每天自动发送 AI 知识摘要到你的邮箱。

---

## 🔧 配置步骤

### 第一步：开启 QQ 邮箱 SMTP 服务

1. 登录 QQ 邮箱网页版：https://mail.qq.com
2. 点击顶部 **设置** → **账户**
3. 找到 **POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务**
4. 点击 **开启服务**（选择 SMTP）

   ![示意图]
   ```
   ☑️ POP3/SMTP服务  [已开启]
   ☑️ IMAP/SMTP服务  [已开启]
   ```

### 第二步：获取授权码

1. 点击 **生成授权码**
2. 需要用手机发送短信验证
3. 验证后会显示 16 位授权码（如：`abcdxyz123456789`）
4. **立即复制保存**，只显示一次！

> ⚠️ **注意**：不是 QQ 密码！是专门的 16 位授权码！

### 第三步：填入配置文件

编辑 `.env` 文件：

```env
# SMTP 配置
SMTP_HOST=smtp.qq.com      # QQ 邮箱服务器地址
SMTP_PORT=587              # QQ 邮箱端口（必须填 587）
SMTP_USER=123456@qq.com    # 你的 QQ 邮箱地址
SMTP_PASSWORD=abcdxyz123456789  # 刚才复制的 16 位授权码
EMAIL_TO=123456@qq.com     # 接收邮件的邮箱（可以和发送邮箱相同）
```

## ❓ 常见问题

### Q1: 显示 "授权码错误"

**解决**：
- 检查是否复制了完整的 16 位授权码（不含空格）
- 重新生成授权码（旧的可能过期）
- 确认 SMTP 服务已开启

### Q2: 显示 "连接失败"

**解决**：
- 检查网络连接
- 确认防火墙没有阻止 587 端口
- 尝试使用 TLS 加密

### Q3: 邮件发送成功但收不到

**解决**：
- 检查垃圾邮件文件夹
- 确认 EMAIL_TO 邮箱地址正确
- QQ 邮箱可能在"收件箱"上方显示

### Q4: 不想用 QQ 邮箱？

**其他邮箱配置：**

| 邮箱 | SMTP_HOST | SMTP_PORT |
|------|-----------|-----------|
| QQ 邮箱 | smtp.qq.com | 587 |
| 163 邮箱 | smtp.163.com | 587 |
| Gmail | smtp.gmail.com | 587 |
| Outlook | smtp.office365.com | 587 |
| 企业微信 | smtp.exmail.qq.com | 587 |

## 📋 最小配置示例

如果只用 QQ 邮箱：

```env
# AI 配置（至少填一个）
OPENAI_API_KEY=sk-your-openai-key

# 邮件配置（QQ 邮箱）
SMTP_HOST=smtp.qq.com
SMTP_PORT=587
SMTP_USER=123456@qq.com
SMTP_PASSWORD=abcdefghijklmnop  # 16位授权码
EMAIL_TO=123456@qq.com

# 定时发送时间（每晚22点）
EMAIL_SCHEDULE=0 22 * * *
```

## 🔐 安全提示

1. **不要将 .env 文件提交到 Git！**
   ```bash
   # 已添加到 .gitignore
   echo ".env" >> .gitignore
   ```

2. **授权码只显示一次**，丢失需要重新生成

3. **定期更换授权码**（建议每 3 个月）
