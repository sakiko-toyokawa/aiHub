# AI Knowledge Hub - 启动指南

## 🚀 快速启动（推荐）

我们提供了三种一键启动方式，无需 Docker：

### 方式一：Python 脚本（推荐，跨平台）

```bash
# 进入项目目录
cd ai-knowledge-hub

# 一键启动
python start.py

# 跳过依赖安装（已安装过时）
python start.py --skip-install

# 只启动后端
python start.py --backend-only

# 只启动前端
python start.py --frontend-only
```

### 方式二：PowerShell 脚本（Windows 推荐）

```powershell
# 进入项目目录
cd ai-knowledge-hub

# 一键启动
.\start.ps1

# 跳过依赖安装
.\start.ps1 -SkipInstall

# 只启动后端
.\start.ps1 -SkipFrontend

# 只启动前端
.\start.ps1 -SkipBackend
```

### 方式三：批处理脚本（Windows 简单版）

```cmd
# 双击运行或命令行执行
start.bat
```

首次启动前，请配置 `backend/.env` 文件：

```env
MINIMAX_API_KEY=your-minimax-key

# 使用 SQLite（最简单，无需安装数据库）

# 或使用 PostgreSQL（需安装数据库）
# DATABASE_URL=postgresql://user:password@localhost:5432/ai_knowledge

# ========== 爬虫配置（可选）==========
# GitHub Token（可选，用于更高频率的 API 访问）

# Bilibili SESSDATA（可选，用于获取登录内容）

SMTP_PASSWORD=your-16-char-auth-code

```

## 📖 使用流程

### 1. 首次启动

```bash
python start.py
```

脚本会自动：
- 检查 Python 和 Node.js 环境
- 创建 Python 虚拟环境 (`backend/.venv`)
- 安装后端依赖
- 安装前端依赖
- 启动后端服务 (http://localhost:8000)
- 启动前端服务 (http://localhost:5173)

### 2. 日常启动

```bash
# 跳过已安装的依赖，直接启动
python start.py --skip-install
```

### 3. 访问应用

- **前端界面**：http://localhost:5173
- **API 文档**：http://localhost:8000/docs
- **健康检查**：http://localhost:8000/api/health

## 🔧 手动启动（高级用户）

如果你想手动控制每个步骤：

### 启动后端

```bash
cd backend

# 创建虚拟环境（首次）
python -m venv .venv

# 激活虚拟环境
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 启动服务
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 启动前端

```bash
cd frontend

# 安装依赖（首次）
npm install

# 启动开发服务器
npm run dev
```

## ❓ 常见问题

### Q: 端口被占用怎么办？

**后端端口被占用：**
```bash
# 修改启动命令的端口
uvicorn app.main:app --reload --host 0.0.0.0 --port 8001
```

**前端端口被占用：**
编辑 `frontend/vite.config.ts`，修改 `server.port`。

### Q: 如何更新依赖？

```bash
# 后端
cd backend
.venv\Scripts\activate  # Windows
pip install -r requirements.txt --upgrade

# 前端
cd frontend
npm update
```

### Q: 如何查看日志？

直接查看弹出的命令行窗口，或使用：
```bash
# 后端日志（手动启动时）
uvicorn app.main:app --reload --log-level debug

# 前端日志（手动启动时）
npm run dev -- --debug
```

### Q: 如何停止服务？

直接关闭对应的命令行窗口即可。
